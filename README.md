# Programming Training 2027

Welcome to programming training! We're excited to have you. Please read the directions below to get started with setting up and learning GitKraken!

#### Cloning this Repository:
* Open GitKraken
* Sign in with your Github account
* Click "Clone"
* Select the folder destination
* Paste in the link you have from Github as the URL
* Clone the repo!
#### Contributing Code:
* Right click on "main" and create a new branch named "feat/[your_name]"
* Double click on your new branch to check it out
* Add a comment in Main.java through VS Code (e.g. // Hi, my name is [my name])
* Hit CTRL + S or CMD + S to save
* Return to GitKraken and click "View Change"
* Stage all changes, and write a summary of what you did
* Commit your changes and push!

Congrats on creating your first commit! Let's continue learning about how we can use GitKraken to collaborate on code.
